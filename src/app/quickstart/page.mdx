# Quickstart

This guide will help you get NextStore set up locally and ready to use. Follow these steps to install the project and start using it. {{ className: 'lead' }}

## Prerequisites

Before you begin, ensure you have the following installed on your machine:

- [Git](https://git-scm.com/) (Version control system)
- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (Node package manager, comes with Node.js)
- A code editor like [Visual Studio Code](https://code.visualstudio.com/)

<Note>
  Before you can make requests to the NextStore API, you will need to grab your
  API key from your admin dashboard. You can find it under [Settings &raquo;
  API](#).
</Note>

## Installation Steps

1.  **Clone the `nextstore-admin` repository:**

    Open your terminal and run the following command to clone the NextStore admin repository:

    ```bash
    git clone https://github.com/Joselay/nextstore-admin.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd nextstore-admin
    ```

3.  **Open the code in any text editor or IDE:**

    Open the code in your preferred text editor or IDE. We recommend using [Visual Studio Code](https://code.visualstudio.com/)

4.  **Install dependencies:**

    Run the following command to install the necessary dependencies:

    ```bash
    npm install
    ```

5.  **Create a `.env` file at the project root directory:**

    In the root of your `nextstore-admin` project directory, create a new file named `.env`.

6.  **Paste the following into the `.env` file:**

    ```bash
    STRIPE_WEBHOOK_SECRET=your_stripe_webhook_secret
    FRONTEND_STORE_URL=http://localhost:3000/
    NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
    CLERK_SECRET_KEY=your_clerk_secret_key
    NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
    NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up

    NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/
    NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/

    DATABASE_URL=your_postgresql_database_url

    NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your_cloudinary_cloud_name
    STRIPE_API_KEY=your_stripe_api_key

    ```

7.  **Start the frontend and API server:**

    In one terminal window, start the frontend:

    ```bash
    npm run dev
    ```

    This will start the backend admin on localhost:3001 and the frontend on localhost:3000.

8.  **Clone the `nextstore` repository:**

    Open your terminal and run the following command to clone the NextStore repository:

    ```bash
    git clone https://github.com/Joselay/nextstore.git
    ```

9.  **Navigate to the project directory:**

    ```bash
    cd nextstore
    ```

10. **Install dependencies:**

    Run the following command to install the necessary dependencies:

    ```bash
    npm install
    ```

11. **Open the code in any text editor or IDE:**

    Open the code in your preferred text editor or IDE. We recommend using [Visual Studio Code](https://code.visualstudio.com/)

12. **Create a `.env` file at the project root directory:**

    In the root of your `nextstore` project directory, create a new file named `.env`.

13. **Paste the following into the `.env` file:**

    ```bash
    NEXT_PUBLIC_API_URL=http://localhost:3001/api/{storeId}
    ```

14. **Start the frontend and API server:**

    In one terminal window, start the frontend:

    ```bash
    npm run dev
    ```

    In another terminal window, navigate to the API directory and start the API server:

15. **Access the application:**

   <Note>
     Make sure to run `nextstore` on port `:3000` and `nextstore-admin` on port `:3001`.

    </Note>
    Open your web browser and navigate to view the frontend of NextStore.

        ```bash
        http://localhost:3000
        ```

        The admin panel can be accessed at

        ```bash
        http://localhost:3001
        ```

        to view the frontend of NextStore. The admin panel can be accessed at http://localhost:3001.
